HTML Formatting 📌

We need to use different text format to make our text look better. or define special meaning.

I think we should know them first.

Tag Name Use for
<b> Bold text (we can use strong tag also. BUT! Not recommended )
<strong> Important text
<i> Italic text
<em> Emphasized text
<mark> Marked text
<u> Underlined text
<small> Smaller text
<del> Deleted text
<ins> Inserted text
<sub> Subscript text
<sup> Superscript text

Now It's your time to implement them.
