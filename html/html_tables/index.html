HTML Tables 📌

An HTML table consists of cells inside rows and columns. It helps us organise our data in form of rows and columns.

Example of HTML table


            
                <!DOCTYPE html>
                <html lang="en">
                <head>
                </head>
                <body>
                    <table>
                    <tr>
                    <th>Person</th>
                    <th>Email</th>
                    <th>Car</th>
                    </tr>
                    <tr>
                    <td>Max</th>
                    <td>abc@d.com</td>
                    <td>Honda</td>
                    </tr>
                    <tr>
                    <td>John</td>
                    <td>abc@d.com</td>
                    <td>Benz</td>
                    </tr>
                    </table>
                </body>
                </html>

We use tr, th, and td tags for HTML tables. The above example is the basic way to define a table in HTML.


Now we will look into table headers, table cells and table rows.

Table headers

Table headers are important cells which are present at the beginning of the rows or columns and are used to specify which type of data will go into the following cells of the particular row or column. They are defines using the <th> tag.

The <th> is bold by default. You can change it using CSS too. In the above example the first row consists of the table headers. 

          
            <tr>
            <th>Person</th>
            <th>Email</th>
            <th>Car</th>
            </tr>

Table data

Individual data cell or table cell is represented by <td>

A table cell can contain any type of data like image, text, number, list, link etc.

          
            <tr>
            <td>John</td>
            <td>abc@d.com</td>
            <td>Benz</td>
            </tr>

Table Rows

We use <tr> html tag to define a table row. Our table can have any number of rows as we want. But we must make sure that all the rows must have same number of cells.

Table Styling

Now lets talk about styling our tables.

For styling our tables we use rowspan and colspan to span over multiple rows and multiple columns.

HTML table - Rowspan and Colspan

To make a cell span over multiple rows, we use the rowspan attribute.

To make a cell span over multiple columns, we use the colspan attribute.
